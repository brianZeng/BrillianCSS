结构化地组织CSS规则

如果将HTML中的树状结构对应于CSS中，那么CSS的管理将会更加轻松

1.子元素嵌套
直接将子元素的选择器和规则写入父元素规则中
div($width:100px){
  width:$width;
  >p{
     margin-left:$width/5;
    }
 }
这段规则会被解释为如下两段
 div{
    width:100px;
 }
 div>p{
    margin-left:20px;
 }
2.父元素引用嵌套
对于:hover,::after这样的规则，请用&代替父元素的选择器
 div{
  background:white;
  &:hover{
    background:gray;
   }
  &::before{
    content:'!';
   }
 }
