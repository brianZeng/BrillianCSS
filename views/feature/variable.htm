优雅地应对css中的变化

ChangeCSS语法中变量使用$开头，变量分为两种类型:

1.默认变量
一般位于Selector后的()中,多个变量用;分隔,例如
SELECTOR($fontSize:120%;$color:white){
...
}
2.覆盖变量
在style之间直接声明的变量，将覆盖同名的默认变量
 h2($color:red){
  color:$color;
 }
 $color:blue;
最终结果为: 
h2{
 color:blue
}
对于多个同名的覆盖变量 
$width:20px;
...
div{
   width:$width;
}
$width:40px;

最终结果为:
div{
 width:40px
}
3.变量或者数值之间可以使用运算符号进行运算
{
 margin-right:$floatWidth-20px;
}
4.对于css中的多值属性，例如:border,请使用space隔开
{
 border:$borderWidth solid $borderColor;
}
